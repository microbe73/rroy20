Input forms are more complex, requiring multiple divs. Inside of a form, creating a div with class "grid-container" organzies and centralizes a grid. Inside of the "grid-container" div, another div can be used to specify the spacing and padding between cells. Each cell inside this can be created with another div with a class specifying its cell size



Div classes "success callout" and "alert callout" create long banners. Can be useful with control flow statements to display to a user if data transfer or other backend processes were successful
